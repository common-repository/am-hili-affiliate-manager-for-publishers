== Description ==

AM-HiLi (affiliate manager for publishers) will help you to:

* Minimize the risk for your website of been banned from search engines by masking the affiliate links.
* maximize your revenue by auto-inserting affiliate links into your websites’ posts and pages.
* Have complete overview of your affiliate statistics.
* Manage your income from your affiliates and your expenses with the build-in bookkeeping tool.

= Main features =
* Hide, shorten or cloak any link in 2 clicks while you add or edit page or post.
* 301, 302, 307 redirection code.
* Rel nofollow for masked links.
* Nofollow and noindex for redirects.
* Create customized affiliate links and insert them in your website's posts, pages and widgets using short codes.
* Generate php code for the theme.
* Deactivate and activate links.
* Set one of the links as default to refill deactivated or deleted links.
* Click counter.
* Manage links categories.
* Manage affiliate’s advertisers.
* Filter links by category or advertiser.
* Sort links by name, category, advertiser, date or clicks.
* Multi websites supported. 

= More features in premium =
* Create and manage auto-insert affiliate links.
* Auto-attach masked affiliate links to certain keywords in posts and pages.
* Limit the number of links can be inserted in a single post.
* Limit the number of the same keyword can be attached by affiliate link.
* Prioritize affiliate links.
* Detailed hourly, daily, weekly, monthly and yearly statistics.
* Detailed statistics per category, advertiser, country, IP address.
* Graphic charts.
* Ban certain IP addresses from abusing your affiliate business. 
* Bookkeeping tool.
* Register your income and expenses. From your affiliates or private business.
* Auto insert advertisers’ name in the bookkeeping table to be used as income source.
* Basic report showing your total income, expenses and saldo.
* Filter the basic report by income, expense, date, advertiser.
* Detailed daily, weekly, monthly, quarterly and yearly reports.
* Bookkeeping graphic chart reports.

== Installation ==
= From your WordPress dashboard =

1.	Visit 'Plugins > Add New'.
2.	Search for 'AM-Hili'.
3.	Install AM-HiLi plugin.
4.	Activate the plugin through the 'Plugins' menu in WordPress

= From WordPress.org =
1.	Download and unzip AM-HiLi plugin.
2.	Upload the am-hili folder to the /wp-content/plugins/ directory
3.	Activate the plugin through the 'Plugins' menu in WordPress
